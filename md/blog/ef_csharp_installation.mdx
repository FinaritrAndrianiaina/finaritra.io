Aprés avoir installer .NET 5 SDK nous allons créer une application console:

```bash
dotnet new console -n TutoEF
```

Puis nous installons les packages et outils nécessaires:

```bash
dotnet add tuto.csproj package Microsoft.EntityFrameworkCore

dotnet add tuto.csproj package Microsoft.EntityFrameworkCore.SQLServer

dotnet add tuto.csproj package Microsoft.EntityFrameworkCore.Tools

dotnet tool install --global dotnet-ef
```
